Copyright (c) 2026 TaguchiModels  

本ソフトは、個人が開発・無償公開しているフリーソフトです  

##  📖 目的  
画像ファイルの4つの角をドラッグアンドドロップで  
操作して歪んだ画像に変形させるためのツールです  

##  📖 使い方  
画像ファイルを選択してください（JPG、PNG対応）  
左側のキャンバスで青い丸（●）をドラッグして変形させます  
右側に変形後の画像がリアルタイムで表示されます  
「リセット」で元の位置に戻ります  
「透過PNG保存」で変形後の画像をダウンロードできます
